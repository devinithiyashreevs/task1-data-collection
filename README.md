Trend Analysis and Visualization using Python:
Overview :
TrendPulse is an end-to-end data analysis project that captures and interprets real-time trending stories from the Hacker News API. It focuses on transforming raw data into meaningful insights about popularity and user engagement.

Approach:
The project follows a structured pipeline:
Data collection using APIs,
Data cleaning and preprocessing,
Statistical analysis with Pandas and NumPy,
Visualization through charts and a consolidated dashboard.

Key Outcome:
The analysis highlights that popularity and engagement do not always move together, offering a deeper understanding of how users interact with trending content.

Tech Stack:
Python, Pandas, NumPy, Matplotlib, Requests

Conclusion:
TrendPulse demonstrates a practical approach to handling real-world data  from extraction to insight and reflects the importance of structured analysis in understanding digital trends.
